        #### Installation
        
        * CE_AI_D002 is currently only available on the github's repository and you can install it via pip if u have the folder in your working directory:
        
        ```bash
        pip install -U CE_AI_D002
        ```
        
        * If you prefer, you can clone it and run the setup.py file. Use the following commands to get a copy from GitHub and install all dependencies:
        
        ```bash
        git clone https://github.com/pateldarshan98/Required_file.git
        cd Required_file
        pip install .
        ```
